# 第1章：工序规则强制约束库（AI 修正核心依据）

## 1.1 通用加工工序固定顺序

**标准工序链 — 必须严格遵守，任何自动生成的工序均须对照此链验证：**

```
Step 1  毛坯开粗（Stock Roughing）
          ↓
Step 2  半精铣（Semi-Finish Milling）
          ↓
Step 3  残料铣 / 清角（Rest Milling / Pencil Cleanup）
          ↓
Step 4  底面精铣（Floor Finish Milling）
Step 4' 侧壁精铣（Wall Finish Milling）
          ↓
Step 5  孔加工序列（按位置分组执行）：
          中心钻（Spot Drill）
            ↓ 钻孔（Drilling）
              ↓ 锪孔 / 沉头孔（Countersink / Counterbore）[如有]
                ↓ 铰孔（Reaming）[精度孔]
                  ↓ 攻丝（Tapping）[螺纹孔]
```

**⚠️ AI 最高频错误：** 自动规划常省略 Step 2（半精铣）和 Step 3（残料铣），
直接从 Step 1 跳到 Step 4，导致余量不均匀、精铣刀过载、表面质量差。

---

## 1.2 工序强制约束规则（Hard Rules — HR）

> 以下规则为绝对约束，AI 输出若违反，必须强制修正，不允许例外。

| 规则ID | 触发条件 | 强制动作 | 违反后果 |
|--------|---------|---------|---------|
| **HR-01** | 零件内圆角半径 R < 开粗刀半径（D_rough / 2） | 必须添加残料铣（Rest Milling）工序，使用更小刀具 | 过多残料积压，精铣刀入刀瞬间过载崩刃 |
| **HR-02** | 壁厚 ≤ 3mm 的薄壁特征 | 轴向切深 ap 必须 ≤ 0.2mm；分层精铣；采用对称交替走刀 | 薄壁振动变形，形位公差超差报废 |
| **HR-03** | 腔深 > 5 × 刀具直径（深腔） | 必须在精铣前执行等高轮廓（Contour）半精加工陡峭侧壁 | 残料不均匀导致精铣颤振，侧壁有振纹 |
| **HR-04** | 执行任何精铣工序前 | 该区域必须已完成开粗 + 半精铣，余量均匀 ≤ 0.25mm | 余量过大导致精铣刀崩刃或尺寸不稳定 |
| **HR-05** | 执行任何钻孔工序前 | 所有孔位必须已执行中心钻（Spot Drill） | 钻头游移定位偏差，位置度超差 |
| **HR-06** | 执行攻丝前 | 底孔必须已钻完，孔径符合底孔要求表，沉头孔已完成 | 丝锥折断，孔损工件报废 |
| **HR-07** | 不锈钢 / 硬钢精铣侧壁 | 必须使用顺铣（Climb Milling）；禁止逆铣精铣 | 逆铣产生加工硬化层，下刀面质量恶化 |
| **HR-08** | 球刀精铣曲面前 | 必须用平底刀/圆鼻刀完成半精铣，余量均匀控制在 0.15~0.2mm | 球刀直接精铣余量不均区域，点接触力集中崩刃 |
| **HR-09** | 开粗后余量 > 1mm 就进行精铣 | 必须插入半精铣工序，将余量减至 0.15~0.25mm | 精铣无法保证尺寸精度和表面粗糙度 |
| **HR-10** | 键槽铣削下刀 | 禁止直接轴向垂直插铣；必须使用螺旋下刀或斜线切入 | 键槽铣刀底部切削能力弱，直插导致刀具折断 |
| **HR-11** | 不锈钢加工中退刀 | 退刀前主轴不得在材料内停转；必须边走边退 | 停转点产生摩擦热，工件局部硬化或刀具烧损 |
| **HR-12** | 转速设置 | 所有计算转速必须钳制在 ≤ 8000 RPM 上限 | 超限报警停机，或主轴超速损坏 |

## 1.3 余量分配标准

| 工序阶段 | 侧面余量（径向） | 底面余量（轴向） | 适用说明 |
|---------|--------------|--------------|---------|
| 开粗后 | 0.4 ~ 0.8 mm | 0.3 ~ 0.6 mm | 铝件取小值；钢/不锈钢取大值 |
| 半精铣后 | 0.15 ~ 0.25 mm | 0.1 ~ 0.2 mm | **均匀余量是精铣质量的前提** |
| 残料铣后 | 0.15 ~ 0.2 mm | 0.1 ~ 0.2 mm | 与半精铣余量保持一致 |
| 精铣后 | 0 mm | 0 mm | 到尺寸，不留余量 |

**⚠️ 不锈钢专项强制注意：**
- **禁止逆铣精铣**（产生加工硬化层，后续工序尺寸漂移）

# 第3章：Fusion 360 CAM 专项规范

## 3.1 加工策略选用矩阵

| 应用场景 | ✅ 推荐策略 | ❌ 不推荐 | 关键控制参数 |
|---------|------------|---------|------------|
| 大体积毛坯开粗 | **Adaptive Clearing（自适应清除）** | Pocket（效率低30%+） | Optimal Load = 0.2~0.45×D |
| 平面 / 顶面去料 | **Face Milling（面铣）** | Parallel（刀纹方向不佳）| Stepover = 60~75%×D |
| 陡峭侧壁半精/精铣 | **Contour（等高轮廓）** | Parallel（留扇贝纹）| Stepdown 半精0.2~0.3mm；精0.05~0.15mm |
| 平底腔底面精铣 | **Pocket / Floor（平行精铣）** | Contour（仅适合壁）| Stepover = 50~60%×D |
| 浅平缓曲面精铣 | **Parallel（平行铣）** | Contour | Stepover = 0.1~0.3mm（依 Ra 要求）|
| 复杂曲面精铣 | **Parallel + Contour 组合** | 单独使用任一 | 平行主切，等高收边 |
| 内圆角 / 清根 | **Rest Machining（残料铣）** | 手动补刀 | 参考前序刀具直径自动计算 |
| 笔式清角（尖角） | **Pencil（铅笔铣）** | 残料铣（步距太大）| 单遍；仅适合外圆角清根 |
| 标准孔位 | **Drilling（钻孔）** | 铣削替代（精度差）| L/D > 5 必启用 Peck Drilling |
| 精度孔（H7/H8） | **Bore / Reaming（铰孔）** | 单纯钻孔 | 底孔留 0.1~0.2mm 铰削余量 |
| 小径螺纹（硬材料）| **Thread Milling（螺纹铣）** | 攻丝（丝锥折断风险）| 螺距必须精确匹配 |

## 3.2 Adaptive Clearing（自适应清除）关键参数设置

```
【开粗标准设置参考】

Optimal Load（最优切削宽度）：
  铝合金     → 0.30~0.45 × D（可激进）
  碳素钢     → 0.20~0.30 × D
  不锈钢     → 0.15~0.25 × D

Maximum Roughing Stepdown（最大轴向步距）：
  铝合金     → 0.5~1.5 × D
  碳素钢     → 0.3~0.5 × D
  不锈钢     → 0.2~0.4 × D

Stock to Leave（留余量）：
  侧面（Radial Stock）：
    钢 / 不锈钢  → 0.5~0.8 mm
    铝合金       → 0.3~0.5 mm
  底面（Axial Stock）：
    钢 / 不锈钢  → 0.4~0.6 mm
    铝合金       → 0.2~0.4 mm

Feed Rate 说明：
  Fusion 360 自动计算 Vf = fz × Z × n
  手动输入 fz 和 n（或 Vc），其余自动换算
```

## 3.3 Contour（等高轮廓）关键参数设置

```
【侧壁半精/精铣标准设置参考】

Stepdown（层间步距）：
  半精铣：0.15~0.30 mm
  精铣  ：0.05~0.15 mm（表面质量 Ra1.6 要求取下限）

Stock to Leave（留余量）：
  半精铣：侧面 0.15~0.25 mm，底面 0.1~0.2 mm
  精铣  ：0 mm（到尺寸）

Direction（铣削方向）：
  ✅ 精铣必须选 Climb（顺铣）
  ❌ 禁止选 Conventional（逆铣）用于精铣钢/不锈钢

Minimum Cutting Radius：
  设为 0，避免内轮廓缺失

Rest Machining 联动：
  勾选 "Use Rest Machining" → 参考前序余量，避免空切
```

## 3.4 残料铣（Rest Machining）触发规则与设置

**触发判断逻辑：**
```
IF  零件内圆角半径 R_corner < 开粗刀半径 R_rough（= D_rough / 2）
THEN  必须执行残料铣

示例：
  开粗刀 ø16mm → R_rough = 8mm
  零件内圆角 R5 < 8mm
  → 结论：必须使用 ø8mm 或更小刀具进行残料铣

残料铣刀具选择：
  残料铣刀直径 = (内圆角半径 R_corner × 2) × 0.8  [取整到标准刀具]
  例：R5 内角 → 使用 ø8mm 刀（5×2×0.8=8）
```

**Fusion 360 设置步骤：**
```
1. 新建刀路（刀具直径 < 开粗刀 2~4mm）
2. Passes 选项卡 → 勾选 "Use Rest Machining"
3. Rest Machining Source → 选 "From previous operation(s)"
4. 余量设置与半精铣相同（0.15~0.25mm）
5. 策略选 Adaptive Clearing 或 Contour（按特征选）
```

---

## 3.5 毛坯设置（Stock Setup）规范

```
毛坯创建方式选择：
  固定尺寸毛坯（棒料/板料）    → Fixed Box / Fixed Cylinder
  前工序加工后的毛坯（翻面等） → From Previous Setup（必须！）
  按 3D 模型外扩              → From Solid + Offset

毛坯偏移量（Stock Offset）建议：
  侧面：1.0~3.0 mm（按实际原材料留量）
  顶面：0.5~2.0 mm
  底面：0（通常已装夹定位）

⚠️ 重要：翻面二次装夹时，若不使用 "From Previous Setup"，
   Fusion 将按原始毛坯计算，导致已加工面被重复开粗，浪费刀具。
```

---

## 3.6 碰撞检查强制规范

```
刀具伸出量（悬伸）限制：
  腔深 / 特征深度 < 20mm   → 悬伸 ≤ 3×D（标准刀柄）
  20~40mm                  → 悬伸 ≤ 4×D（长刃或长颈刀）
  40~80mm                  → 悬伸 ≤ 5×D，切削参数降低 30%
  > 80mm                   → 必须专用长颈刀，参数再降 50%，仿真确认

碰撞检查步骤（每次刀路生成后必做）：
  1. 选中工序 → Simulate
  2. 勾选 "Tool Holder"（刀柄） + "Fixture"（夹具）+ "Stock"（毛坯）
  3. 出现红色碰撞警告 → 禁止输出NC，必须调整刀具伸出长度或修改刀路高度
  4. 确认无碰撞后方可 Post Process
```

# 第4章：典型零件工艺路线库

## 4.1 模具型腔（Mold Cavity）

```
工件材料：P20 / 718 模具钢（预硬 ≤38HRC）
典型特征：深腔 + 复杂曲面 + 多内圆角
工艺目标：尺寸公差 ±0.05mm，Ra ≤ 1.6μm

─────────────────────────────────────────────────
Step 1  【开粗】Adaptive Clearing
  刀具：ø20 4刃立铣刀（TiAlN涂层）
  n=1432 RPM（Vc=90），fz=0.04mm，Vf=229 mm/min
  Optimal Load=5mm（0.25×D），ap=8mm（0.4×D）
  余量：侧 0.6mm，底 0.5mm

Step 2  【型腔壁半精铣】Contour
  刀具：ø12 4刃立铣刀（TiAlN涂层）
  n=2920 RPM（Vc=110），fz=0.025mm，Vf=292 mm/min
  Stepdown=0.25mm，余量：0.2mm，顺铣

Step 3  【残料铣内圆角】Rest Machining（触发：R<10mm的内圆角）
  刀具：ø8 4刃立铣刀，参考ø20开粗余量
```

## 4.5 薄壁异形件（Thin-Wall Irregular Part）

```
工件材料：铝合金 / 不锈钢，壁厚 1~3mm，高径比大

强制工艺约束（比标准工序更严格）：
─────────────────────────────────────────────────
开粗阶段：
  ① 按对称方向交替开粗（非单侧连续切削）
  ② ap 比正常减 50%（铝：ap ≤ 0.4×D；钢：ap ≤ 0.2×D）
  ③ 切削力偏向防变形：Optimal Load 降低 20%

半精铣阶段：
  ① ap ≤ 0.3mm，余量保持 0.2mm
  ② 使用等高轮廓（Contour），从底部向上分层

精铣阶段（最关键，必须执行以下所有步骤）：
  ① 分层精铣：每次 ap = 0.1~0.15mm
  ② 对称精铣：两侧交替走刀（切A面一层→切B面一层→循环）
  ③ 支撑工艺：蜡填充空腔 或 低熔点合金灌注
  ④ 使用减振刀柄（HSK 接口 + 阻尼刀杆）
  ⑤ 最终精铣：顺铣 + fz 降低 30% 以减小切削力

孔加工（最后执行，此时零件刚性最差）：
  同侧孔位一次性连续完成；不要跨侧切换
─────────────────────────────────────────────────
```

# 第5章：孔加工专项规范

## 5.1 孔加工标准工序链

```
所有孔加工必须严格按以下顺序：

① 中心钻（Spot Drill）  ← 禁止省略（HR-05）
   刀具：ø3~ø5 中心钻（90° 或 120° 锥角）
   深度：仅打引导坑（深度 ≈ 孔径 × 0.1~0.2）
   目的：防止钻头游走，保证位置精度

② 钻孔（Drilling）
   底孔直径见下方速查表
   L/D > 5 时必须启用啄钻（Peck Drilling），每啄 0.5~1×D 退出排屑
   不锈钢钻孔进给为铝件 40~50%，使用专用不锈钢钻头

③ 铰孔（Reaming）  ← 精度孔（H7/H8 级别）才用
   铰削余量：0.1~0.2mm（底孔孔径 = 铰孔直径 - 0.15mm）
   Vc = 钻孔 Vc × 0.6，进给加倍
   禁止逆转退出铰刀（会刮伤孔壁）

④ 锪孔 / 沉头孔（Countersink / Counterbore）
   在底孔完成后进行，直径精确匹配螺钉规格

⑤ 攻丝（Tapping）  ← 必须最后执行（HR-06）
   在 Fusion 360 中选 "Tapping" 策略
   螺距必须与丝锥规格完全一致（不可估算）
   使用刚性攻丝（机床需支持主轴同步进给）
   不锈钢攻丝转速降至碳钢 50%，使用含硫攻丝油
```

# 第6章：AI 编程错误修正规则库（Negative Correction Rules）

## 6.1 错误代码索引表

| 错误代码 | 错误描述 | AI 典型输出 | 检测条件 | 强制修正动作 |
|---------|---------|-----------|---------|------------|
| **ERR-01** | 缺失半精铣 | 开粗后直接排精铣 | 工序中无半精铣 Operation | 在精铣前插入 Contour 半精铣，余量设 0.2mm |
| **ERR-02** | 缺失残料铣 | 内圆角未清角 | R_corner < D_rough/2 且无 Rest Machining | 添加残料铣，刀具直径 ≤ R_corner × 1.6 |
| **ERR-03** | 工序顺序颠倒 | 精铣在开粗之前 | 检查 Operation 排列顺序 | 强制按第1章工序链重排所有 Operation |
| **ERR-04** | 余量不合理 | 精铣余量设为 0.8mm 或负值 | 余量 > 0.3mm（精铣）或 < 0mm | 按第1章余量分配表统一修正 |
| **ERR-05** | 孔加工缺中心钻 | 直接排钻孔 Operation | 无 Spot Drill 在钻孔之前 | 所有钻孔前插入中心钻工序 |
| **ERR-06** | 孔加工顺序错误 | 攻丝在钻孔前，或无底孔 | 检查孔加工操作顺序 | 修正为：中心钻→钻孔→锪孔→攻丝 |
| **ERR-07** | 逆铣精铣 | Direction 设为 Conventional | 精铣钢/不锈钢使用逆铣 | 强制改为 Climb（顺铣）|
| **ERR-08** | 刀具选型错误 | 铝合金开粗用4刃刀 | 铝合金工序使用4刃立铣刀 | 更换为 2~3 刃铝专用立铣刀 |
| **ERR-09** | 转速超过机床限制 | n 计算值 > 8000 RPM | 自动计算 n = Vc×1000/(π×D) | 钳制 n ≤ 8000 RPM，反算实际 Vc |
| **ERR-10** | 薄壁大切深 | 薄壁件 ap = 2mm | 壁厚≤3mm 且 ap > 0.3mm | 强制设 ap ≤ 0.2mm，增加分层数 |
| **ERR-11** | 深腔无排屑策略 | 腔深>5D，钻孔无啄钻 | 深腔 L/D > 5 且 Peck Drill 未启用 | 启用 Peck Drilling，每啄深 0.5~1×D |
| **ERR-12** | 球刀开粗 | 用球刀执行开粗工序 | Roughing 工序刀具为球刀 | 更换为平底立铣刀；球刀仅用于曲面精铣 |

# 附录A：常用公式与单位换算

```
转速（RPM）：
  n = (Vc × 1000) / (π × D)
  Vc: m/min | D: mm | n: RPM
  示例：ø12铝合金开粗，Vc=230 → n = (230×1000)/(3.14159×12) = 6104 RPM

线速度：
  Vc = (n × π × D) / 1000

进给率：
  Vf = fz × Z × n
  fz: mm/tooth | Z: 刃数 | Vf: mm/min

材料去除率：
  MRR = Vf × ae × ap
  单位：mm³/min

刀具寿命参考（Taylor 近似）：
  Vc 每增加 10%，刀具寿命约降低 50%
  反之，Vc 降低 10%，寿命约延长 100%
```
